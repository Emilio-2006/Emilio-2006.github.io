The SDLC phases

Idea Phase

The idea phase is about people thinking about their project. Also think about what they want in the app.

Money phase

The money phase is used to see how much the project will be. Also how much they will pay the amount of people that they need to make the project.

Designing phase

The designing phase is about making the design and what's on the project and what's in it. Then once the design is done they pass it down to the planning phase.

Planning phase

The planning phase is about the people who make the plan for the project and then when the entire plan is done they will pass it to the coder then the coder will code everything for the app or project.

Coding Phase

The coding phase is about the people who code the project to see if everything is going good in the app or project. If something is wrong then the coders will pass it back to the planner.

Testing Phase

The testing phase is about the people who test the project and see if there are any mistakes and if there are they will tell the coders that something is wrong in the code then the coders will have to figure out what is wrong in the code.

Deploying Phase

The deployment phase is about the people who take the final project once everything is done and the app is complete. Then they move the completed code to a place where the user can access the program.

Job Roles

Product Manager:

makes the big disicions about the product and what the users think should be in the product

CEO-Cheif Executive Officer:

is the boss

Graphic Designers:

create visual concepts, using computer software or by hand, to communicate ideas that inspire, inform, and captivate consumers.

Project Manager:

is the person who keeps track of the plan, makes the plan, and removes impediments

Applications Software Developer:

is a person who designs computer applications such as word processors, and games for consumers.

Web Developer:

is a person who creates websites

Data Engineer:

clears data to controll data bases

Mobile Developer:

is a person who creates mobile applications.

Data Architect:

is an information technology specialist who designs and manages data systems

Test Manager:

The people responsible for the planning, coordination, and control of various test activities.

Release Engineer:

is a person who takes the whole app and releases it into the technology world.
